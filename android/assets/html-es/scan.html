Para escanear, simplemente llevar el código de barras a la vista en el centro de la pantalla del dispositivo. Barcode Scanner+ Simple intentará detectar automáticamente y descodificar el código de barras, que no es necesario pulsar un botón.

Verás puntos amarillos aparecen en la pantalla. Estos indican que la aplicación cree que ha encontrado parte de un código de barras en la imagen en la pantalla. Cuando se decodifica un código de barras con éxito completo, se oye un pitido y / o vibración, y la pantalla de resultados pantalla. (Usted puede activar o desactivar el sonido y las vibraciones en Configuración.)

La pantalla de resultados se muestran información adicional de los productos , Y se presentarán los botones de acción que se pueden utilizar para actuar en el contenido del código de barras .

No mantenga el código de barras demasiado cerca de la cámara, no puede centrarse tan cerca. El código de barras no deben ocupar toda la pantalla. Usted verá una imagen de ejemplo de código de barras transparentes en el centro de la pantalla. Alinear el código de barras de esta guía para asegurarse de que está en la distancia sugerida.

Al escanear formatos cuadrados, en 2D como Código QR, Data Matrix y los aztecas, se alinean con el ejemplo cuadrados en 2D que se ve en color azul claro. Al escanear toda formatos 1D como UPC y códigos EAN de productos, se alinean con el ejemplo de código de barras 1D más amplio en color de rosa. Estas dos guías se mostrará de forma alterna.

Cámaras delantera

Algunos dispositivos, como los Tablet PCs, tiene una cámara frontal adicional. Seleccione "Preferir cámara frontal" en Configuración para escanear con él. Por lo general, sólo es posible escanear los formatos 2D con esta cámara.

Blanco sobre negro de códigos de barras

Los códigos de barras suelen ser impresos en negro sobre un fondo blanco. Algunos, sin embargo, se imprimen en blanco sobre negro. Para analizar estos, permiten "Invertir scan" en Configuración. Usted debe ver una imagen de vídeo invertido. Tenga en cuenta que esta función no tiene ningún efecto en algunos dispositivos.

Escaneo con poca luz

Es difícil para escanear códigos de barras con poca luz o irregular. Algunos dispositivos tienen una luz, que se utiliza como un flash para tomar fotografías. Esto se puede activar en forma continua para iluminar el código de barras que se escanea. Seleccione "Utilizar la luz delantera" en Configuración para permitir que la luz, o pulse la tecla Subir volumen en el dispositivo si tiene una. No todos los dispositivos tienen una luz que se puede habilitar. Tenga en cuenta que la luz puede causar deslumbramiento, que también hace que el escaneo difícil.

Usando el enfoque automático

Por defecto, la aplicación utilizará el dispositivo de cámara de enfoque automático de características, cuando estén disponibles, con el fin de escanear códigos de barras mejor. Esto se debe dejar habilitado, en general. Sin embargo, es posible desactivar el auto-enfoque desmarcando "Usar el enfoque automático" en la configuración.
