Barcode Scanner+ Simple can scan many other formats, which are less common but still used in specific contexts like travel tickets or industrial part labeling. Scanning for each of these formats can be enabled or disabled in Settings; they are disabled by default.

PDF417

PDF417 is usually used by package delivery companies to encode shipping information. Note that it is difficult to read PDF417 from a mobile device. You can choose to enable high resolution preview in Settings, which will improve decoding of this format.

Aztec

Aztec barcodes are primarily found on tickets for trains and airplanes.

Industrial 1D formats

Finally, Barcode Scanner+ Simple can read several different 1D formats that are used primarily in commercial or industrial applications. Above is an example of Code 39; they can be hard to distinguish visually. The application can read:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

In particular, it can display some information about VIN (Vehicle Identification Number) barcodes.
