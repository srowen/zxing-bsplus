Barcode Scanner+ Simple może skanować wiele innych formatów, które są mniej popularne, ale nadal stosowane w specyficznych sytuacjach, takich jak bilety podróżne lub przemysłowych oznakowania części. Skanowanie dla każdego z tych formatów można włączyć lub wyłączyć w Ustawienia, są osób niepełnosprawnych domyślnie.

PDF417

PDF417 jest zwykle stosowany przez firmy o przesyłkach do kodowania informacji wysyłki. Zauważ, że jest trudny do odczytania PDF417 z urządzenia przenośnego. Możesz wybrać, aby umożliwić podgląd wysokiej rozdzielczości w ustawieniach, co poprawi dekodowanie tego formatu.

Aztec

Aztec kody kreskowe są przede wszystkim znaleźć na bilety na pociągi i samoloty.

Przemysłowe 1D formatów

Wreszcie, Barcode Scanner+ Simple można przeczytać kilka różnych formatów 1D, które są wykorzystywane przede wszystkim w zastosowaniach przemysłowych. Powyżej znajduje się przykład Code 39, mogą być trudne do odróżnienia wizualnie. Aplikacja może być:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

W szczególności, może wyświetlać informacje o VIN (numer identyfikacyjny pojazdu) kodów kreskowych.
