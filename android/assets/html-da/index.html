Denne version fjerner eller deaktiverer de fleste funktioner, der kræver særlige tilladelser, herunder: Send historie, send stregkode, scanne fra fil, konfigurere wi-fi, scan NFC-tags, og vibrationer.

Barcode Scanner+ Simple bruger kameraet på din enhed til at læse stregkoder af alle slags. Efter scanning af stregkoder du finder på produkter, for eksempel kan du opdage priser og anmeldelser for det. Eller, når du scanner nyere to-dimensionelle stregkoder, ligesom QR koder, kan du få adgang til kontaktoplysninger, bogmærker, web site links og mere inde.
