Barcode Scanner+ Simple kan scanne mange andre formater, som er mindre almindelige, men stadig bruges i bestemte sammenhænge som rejsehjemmel eller industrielle del mærkning. Scanning for hver af disse formater kan aktiveres eller deaktiveres i Indstillinger, de er deaktiveret som standard.

PDF417

PDF417 normalt bruges af pakkelevering virksomheder til at indkode leveringsoplysninger. Bemærk at det er vanskeligt at læse PDF417 fra en mobilenhed. Du kan vælge at aktivere høj opløsning eksempel under Indstillinger, som vil forbedre afkodning af dette format.

Aztec

Aztec stregkoder er primært fundet på billetter til tog og fly.

Industriel 1D formater

Endelig, Barcode Scanner+ Simple kan læse flere forskellige 1D formater, som primært anvendes i kommercielle eller industrielle applikationer. Ovenfor er et eksempel på Code 39, de kan være svære at skelne visuelt. Ansøgningen kan læse:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

I særdeleshed kan den vise nogle oplysninger om VIN (Vehicle Identification Number) stregkoder.
