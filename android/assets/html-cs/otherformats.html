Barcode Scanner+ Simple může skenovat mnoho dalších formátů, které jsou méně časté, ale stále používá ve specifických situacích, jako jízdenky nebo průmyslové části označení. Skenování pro každý z těchto formátů lze zapnout nebo vypnout v nastavení, jsou zdravotním postižením ve výchozím nastavení.

PDF417

PDF417 se obvykle používají firmy balíček dodání k zakódování informace přepravy. Všimněte si, že je obtížné číst PDF417 z mobilního zařízení. Můžete si vybrat, aby s vysokým rozlišením náhled v nastavení, která zlepší dekódování tohoto formátu.

Aztec

Aztécké čárové kódy jsou primárně k dispozici na jízdenky pro vlaky a letadla.

Průmyslové 1D formáty

A konečně, Barcode Scanner+ Simple můžete přečíst několik různých formátů 1D, které se používají především v komerčních a průmyslových aplikacích. Nahoře je příklad toho, Code 39, které může být obtížné rozlišit vizuálně. Aplikace umí číst:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

Zejména může zobrazit některé informace o čárových kódů VIN (Vehicle Identification Number).
