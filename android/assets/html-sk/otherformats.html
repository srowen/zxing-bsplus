Barcode Scanner+ Simple môže skenovať mnoho ďalších formátov, ktoré sú menej časté, ale stále používa v špecifických situáciách, ako lístky alebo priemyselné časti označenia. Skenovanie pre každý z týchto formátov je možné zapnúť alebo vypnúť v nastaveniach, sú zdravotným postihnutím v predvolenom nastavení.

PDF417

PDF417 sa obvykle používajú firmy balíček dodaní k zakódovanie informácie prepravy. Všimnite si, že je ťažké čítať PDF417 z mobilného zariadenia. Môžete si vybrať, aby s vysokým rozlíšením náhľad v nastavení, ktorá zlepší dekódovanie tohto formátu.

Aztec

Aztéckej čiarové kódy sú primárne k dispozícii na lístky pre vlaky a lietadlá.

Priemyselné 1D formáty

A konečne, Barcode Scanner+ Simple môžete prečítať niekoľko rôznych formátov 1D, ktoré sa používajú predovšetkým v komerčných a priemyselných aplikáciách. Hore je príklad toho, Code 39, ktoré môže byť ťažké rozlíšiť vizuálne. Aplikácia vie čítať:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

I synnerhet kan den visa lite information om VIN (Vehicle Identification Number) streckkoder.
