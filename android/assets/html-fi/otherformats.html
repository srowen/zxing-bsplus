Barcode Scanner+ Simple osaa lukea useita harvinaisempiakin viivakoodityyppejä, jotka ovat käytössä erikoisalueilla, kuten matkalipuissa tai teollisuuden (vara)osien etiketöinnissä. Näiden viivakoodityyppien skannaus voidaan ottaa käyttöön tai poistaa käytöstä Asetuksista; ne ovat oletusarvoisesti poissa käytöstä.

PDF417

PDF417 -koodausta käyttävät yleensä kuriiripalveluyhtiöt, jotka koodaavat viivakoodiin kuljetukseen tarvittavaa tietoa. Huomaa, että PDF417:n lukeminen mobiililaitteella on vaikeaa. Voit valita Asetuksista päälle esikatselun korkean tarkkuuden, mikä parantaa tämäntyyppisen viivakoodin lukemista.

Aztec

Aztec -viivakoodeja käytetään pääasiassa lento- ja junalipuissa.

Industrial 1D -viivakoodit

Lopuksi, Barcode Scanner+ Simple voi lukea monia erilaisia 1D -tyyppisiä viivakoodeja, jotka ovat pääasiassa käytössä kaupallissa ja teollisissa sovelluksissa. Ylhäällä on esimerkki Code 39 -viivakoodista; niitä voi olla vaikea erottaa silmämääräisesti. Tämä ohjelma osaa lukea seuraavia viivakoodityyppejä:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

Erityistä: ohjelma osaa näyttää joitain tietoja VIN-viivakoodista (Vehicle Identification Number, eli Ajoneuvotunnusnumero).
